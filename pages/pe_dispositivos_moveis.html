Situação de Aprendizagem

Desafio: App "GastroAvaré Finder" com React Native

Vocês são uma equipe de desenvolvedores da "InovaApp Solutions", uma startup contratada para criar um aplicativo móvel para o evento "GastroAvaré", um festival de food trucks. O objetivo é ajudar os visitantes a localizar os food trucks, ver seus cardápios e receber notificações sobre eventos especiais.

Sua equipe deverá:

  • Planejar a arquitetura do aplicativo (telas, navegação e componentes).
  • Configurar o ambiente de desenvolvimento com React Native e Expo.
  • Desenvolver as interfaces de usuário (UI) para a lista, detalhes e mapa.
  • Implementar a lógica para consumir dados de uma API RESTful.
  • Integrar APIs nativas de mapas e localização através do Expo.
  • Implementar persistência de dados local para salvar favoritos.
  • Gerar uma versão compilada do aplicativo (APK/IPA) usando Expo Application Services (EAS).
  • Apresentar o aplicativo funcional, testado no app Expo Go.

Recursos à disposição:

  • Computadores com VS Code, Node.js e Expo CLI.
  • Smartphones com o aplicativo Expo Go para teste em tempo real.
  • Documentação de uma API RESTful com os dados do evento.
  • Acesso a serviços de mapas (Google Maps API).
