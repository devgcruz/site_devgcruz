Situação de Aprendizagem

Desafio: Site Informativo "Sala de Aula Saudável"

Vocês são uma equipe de desenvolvedores web encarregada de criar um site informativo para o projeto "Sala de Aula Saudável". O objetivo é promover a saúde e o bem-estar no ambiente educacional, oferecendo dicas e informações para alunos e professores sobre ergonomia, pausas ativas e bem-estar mental. O site precisa ser bem estruturado, acessível e responsivo.

Sua equipe deverá:

  • Estruturar o conteúdo do site (Página Inicial, Ergonomia, Pausas Ativas, Bem-estar, Contato) usando HTML semântico.
  • Estilizar todas as páginas utilizando CSS, com foco em um design calmo e acessível.
  • Implementar o Box Model para criar leiautes de conteúdo bem definidos.
  • Utilizar Flexbox ou Grid para criar um design responsivo que se adapte a diferentes tamanhos de tela.
  • Criar um formulário de contato para sugestões.
  • Publicar o site finalizado em um serviço de hospedagem gratuito.

Recursos à disposição:

  • Computadores com editor de código (VS Code).
  • Navegadores web com ferramentas de desenvolvedor.
  • Conteúdo (textos e imagens) sobre ergonomia, exercícios de alongamento e dicas de bem-estar.
